# AWS-Projects
AWS Project

1a. Created an AWS Glue crawler to scan raw weather data in Amazon S3, infer schemas, and populate the AWS Glue Data Catalog, applying exclusion patterns to optimize crawl performance and organizing discovered tables within a dedicated target database for downstream analytics.

[Video](https://youtu.be/-xUIBH9B-jU)



1b. Built an AWS Glue Studio Visual ETL job to transform raw weather data by renaming and casting date fields, removing unused long-type columns, and writing curated parquet outputs to Amazon S3 with partitioning and automatic Data Catalog updates.

[Video](https://youtu.be/3UgRh75J23s)



1c. Configured Athena query settings and executed analytical SQL queries against partitioned Parquet tables to explore weather trends and verify transformation outputs.

[Video](https://youtu.be/ENkFDAhl1yA)
